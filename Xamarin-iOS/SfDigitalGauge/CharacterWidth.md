# CharacterWidth

The value of the Digital Characters is scaled by altering the width of the digital characters. It is achieved by setting the CharacterWidth property in SfDigitalGauge.

{% highlight c# %}

	digitalGauge.CharacterWidth = 18;

{% endhighlight %}
